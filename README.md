# House Price Prediction using Machine Learning

This project demonstrates how regression models can be used in real-world problems.

## Problem Statement
Predict house prices using Linear Regression and classify houses as expensive or not using Logistic Regression.

## Features Used
- Area (sq ft)
- Number of bedrooms
- Location score

## Models Implemented
- Linear Regression (price prediction)
- Logistic Regression (binary classification)
